# ChatWithEmoji

### Setup Instructions:
* Clone the Repository
    ``` git clone https://github.com/adishap/chatWithEmoji ```

### Run the server:
1. `cd chatWithEmoji`
2. To run the chatWithEmoji
    ``` python -m SimpleHTTPServer 8000 ```
3. For chatWithEmoji, go to [http://localhost:8000](http://localhost:8000)
